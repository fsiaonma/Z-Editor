帮助文档

api接口:

  • 全局对象: OLEditor

  • init (初始化OLEditor方法)

    用例: OLEditor.init();

    参数: 无

  • getValue (获取编辑器文本内容)

    用例: OLEditor.getValue();

    参数: 无

  • setValue (获取编辑器文本内容)

    用例: OLEditor.setValue(data, type);

    参数:

    data: 要填入编辑器的内容

    type: 要填入编辑器内容的类型

    (可选参数3个: 'text/html', 'css', 'javascript')

编辑器功能:

  • Ctrl + F 可以进行搜索

  • F11 可切换到全屏, Esc/F11 退出

  • 根据html代码动态生成效果图

  • 对应不同格式文件高亮

  • 双击某一词语能显示文本中其他相同词语

  • 显示选中行
