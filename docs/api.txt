全局对象 : OLEditor

init : 初始化OLEditor方法
用例 : OLEditor.init();
参数 : 无

getValue : 获取编辑器文本内容
用例 : OLEditor.getValue();
参数 : 无

setValue : 获取编辑器文本内容
用例 : OLEditor.setValue(data, type);
参数 : @params data <String> 要填入编辑器的内容
       @params type <String> 要填入编辑器内容的类型 (可选参数3个: 'text/html', 'css', 'javascript')


编辑器功能
1. Ctrl + F 可以进行搜索
2. F11 可切换到全屏, Esc/F11 退出
3. 根据html代码动态生成效果图
4. 对应不同格式文件高亮
5. 双击某一词语能显示文本中其他相同词语
6. 显示选中行
